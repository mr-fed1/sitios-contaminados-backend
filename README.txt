Proyecto Backend - Sitios Contaminados (MongoDB + Spring Boot)

Instrucciones:
1. Cloná este proyecto o subilo a tu repo de GitHub.
2. Asegurate de tener Maven y JDK 17 instalados.
3. Configurá la variable de entorno MONGODB_URI en Railway (o .env si lo corres local).
4. Ejecutá: mvn clean package -DskipTests
5. Deployá el .jar en Railway o corré localmente con: java -jar target/alertas-mongo-1.0.0.jar
